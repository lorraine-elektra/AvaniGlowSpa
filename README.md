# Avani Glow Spa

Modern luxury spa website built as a frontend portfolio project for a premium wellness brand in Osu, Accra.

## Live Demo
https://avaniglowspa.pages.dev/


## Preview

## Features

- Fully responsive layout
- Luxury-inspired UI/UX
- Interactive gallery filtering
- WhatsApp booking integration
- Promotions & referral sections
- FAQ accordion
- Smooth animations and transitions
- SEO & Open Graph optimization

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
